1.Overview

DREAM --A Toolbox to Decode Rhythms of the Brain System is a MATLAB-based toolbox which could decoding rhythms of the brain to seveal differents frequnency bands which decided by your own data.

2. Running inside Matlab

   Command-line

   You can add DREAM to MATLAB's path with following command:
         >>addpath(genpath('E:\Brainsoft\DREAM'))
   where 'E:\Brainsoft\DREAM' is the location of your DREAM.

   Interface

   1. Click "Add with Subfolders..." button and select your DREAM path. i.e. "E:\Brainsoft\DREAM".
   2. Save your change.

3. Using DREAM

Typing DREAM at the prompt to active DREAM GUI

Click on the GUI to enter the main interface.

4.Input data Format

   Brain Image

   DREAM support fMRI sequences with compressed NIFTI format ending with '.nii.gz'.

   Head Motion Data

   DREAM support head motion data and other file ending with '.1D'.

   Head motion data should be organized into the follow format:
   Roll = rotation about the I-S axis.
   Pitch = rotation about the R-L axis.
   Yaw = rotation about the A-P axis.
   ZdS = displacement in the Superior direction.
   XdL = displacement in the Left direction.
   YdP = displacement in the Posterior direction.
   The *.1D file should be contain 6 lines and n rows where n is your time points.

   EEG Data

   DREAM support EEG data which was recorded with Brain Products. Please check whether your data has 3 files (.vhdr/.eeg/.vmrk) in each subjects.

   Other Format

   DRAM will support decoding MEG and other type of EEG data in a recent update.

5. Main interface

Data Type: Choose your data type. e.g. NIFTI
Working Directory: Choose your working directory. e.g. E:\BrainData
Sublist: Choose a list file which contain all you subjects. e.g. E:\BrainData\sublist.list
TR(s): The time of repeat of data signal. e.g. 2
Data Dir: The name of data directory e.g. /func

After finish data set, the filelist area would list all your data.
You can delete the file by click the 'remove' button.
The right panel will show your data view when you choose the data you want, the title of the view means the frequency range of the data.
Make sure the parameters are set correctly, click 'decode'.

7.Result

DREAM would run and show results on the right panel when finished step 6. The time required depends on the size of your data.


After Decode, some folders is created in each subject's folder, which name is same with your chosen data name(without suffix).
The freqbands.csv contains all frequency bands decoded from your data. In these directories, *FBsn are created.
* is same with your chosen data name(without suffix)
n represent the number of frequency bands.
